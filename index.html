vegasNavigation

A script for implementing navigation on a site with a normal drop-down list and with mega content.

  • home page
  • github
  • npmjs

Install

npm i vgnav

or download here

Include files

CSS. Main file vgnav.css, default theme vgnav-theme.css

<link href="..path../dist/css/vgnav.css" rel="stylesheet" type="text/css">

JavaScript. Main file vgnav.js, or minified vgnav.min.js

<script src="..path../dist/js/vgnav.min.js"></script>

Usage

HTML markup

Required classes:

  • .vg-nav - Main script class
  • .vg-nav-main-container - Root level navigation

Leveled classes:

  • .dropdown - Dropdown menu
  • .dropdown-mega - Dropdown mega menu

Classes indicating direction:

  • .left - The menu will open from left to right
  • .right - The menu will open from right to left

If no class is specified, by default the menu will open from left to right

Markup:

<nav class="vg-nav">
    <ul class="vg-nav-main-container">
        <li class="active"><a href="/">Home page</a></li>
		<li class="dropdown">
            <a href="#">Left dropdown</a>
            <ul class="left">
                ...
            </ul>
        </li>
        <li class="dropdown-mega">
            <a href="#">Mega Menu</a>
            <div class="dropdown-mega-container">
                ...
            </div>
        </li>
        <li class="dropdown">
            <a href="#">Right dropdown</a>
            <ul class="right">
		       ...
            </ul>
        </li>
    </ul>
</nav>

Initialize

new VGNav({
  expand: 'md',
}).toggle()

Attention. The script is initialized once

Options

Name Type Default Description
expand string lg

The parameter specifies at what screen resolution the menu should be minimized

Available values:

  • xl - large screen devices, from 1200px
  • lg - screens with frames from 992px to 1200px
  • md - screens with frames from 768px to 992px
  • sm - screens with frames from 0 to 768px
  • xs - screens with frames from 0 to 480px
layout string sidebar

The parameter specifies at what screen resolution the menu should be minimized

Available values:

  • collapse - place the following markup on the page <div class="vg-nav-collapse"></div>
  • sidebar - installed automatically
isHover boolean false

The expanded menu can be opened not only by clicking, but by hovering
toggle string <span class="default"></span>

You can also customize the arrow that points to the drop-down levels
sidebar object This set includes the following parameters: placement, clone
sidebar.placement string right This parameter specifies the direction of the side panel opening. Available values left, right, top, bottom
sidebar.clone string null Specify in this parameter the identifier (or class) of the block where the menu will be cloned, the parameter is used only in its own markup

And more possibilities

You can also use the custom sidebar markup, for this, in the necessary block, specify the class .vg-nav-sidebar.

In your panel, you can also use the date tag with the data-dismiss="vg-nav-sidebar" parameter to close it.
